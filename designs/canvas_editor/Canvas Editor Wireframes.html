Four layout approaches

Same content, four different chrome arrangements. V1 is the standard three-panel layout. V2 trades the components grid for a thin icon rail with a flyout. V3 removes fixed panels entirely and lets a ⌘K command palette + contextual floating controls drive everything. V4 unifies Add, Layers, and Style into one tabbed left panel.
