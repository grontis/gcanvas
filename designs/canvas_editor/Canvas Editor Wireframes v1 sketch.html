The editor

four layout approaches

Same content, same tools — four different chrome arrangements. V1 is the Webflow/Figma familiar. V2 trades the component grid for a thin icon rail and pops the library as a flyout. V3 removes panels entirely and lets contextual toolbars float over the canvas. V4 unifies components + layers + inspector into one left-side tabbed panel.

Editing a text block

rich-text toolbar on selection

Double-click any text to enter edit mode. A floating toolbar hovers above the cursor with the essentials: size, weight, color, link, list, alignment. The right-side inspector mirrors the block-level typography settings.
